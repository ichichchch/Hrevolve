🔄 Hrevolve
HR that evolves with you.

A lightweight, open-source Human Resource Management system built for growing teams. 

Hrevolve 是專為初創團隊與中小企業設計的輕量級人力資源管理系統。我們相信，HR 工具不該繁瑣沉重，而應隨團隊成長持續進化（evolve）。
從員工資料、出勤管理到人事流程自動化，Hrevolve 提供核心功能，零複雜，極簡部署，快速上線。

🚀 輕 — 無臃腫模組，只保留真正需要的功能
🌱 活 — 架構清晰，易擴展，可自由定制
🔁 進化 — 持續迭代，因應團隊需求而變

✨ 核心特色
員工資料管理（Employee Records）
出勤與請假追蹤（Attendance & Leave）
輕量人事流程（Onboarding / Offboarding）
多角色權限控制（Admin / HR / Employee）
支援 Docker 快速部署
前後端分離，易於二次開發
🛠 技術棧
Frontend: React.js / Tailwind CSS
Backend: Node.js (Express) 或 Python (FastAPI/Django)
Database: PostgreSQL / SQLite
Deploy: Docker, Docker Compose ready
💡 預設為輕量設計，也可依需求擴充薪資、績效等模組。
